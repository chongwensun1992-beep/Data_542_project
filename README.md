# 4. Running the Notebooks

All experiments were executed using Jupyter Lab.  
Each notebook contains a fully self-contained workflow including:
- Data loading
- Data processing and cleaning
- Analysis and visualization
- Export of reproducible results to the `results/` directory

To run the notebooks:
1. Start the environment using `conda activate ai-dev-msr`
2. Launch Jupyter Lab with `jupyter lab`
3. Open and run the notebooks in the following order:
   - `RQ1_testing_behavior.ipynb`
   - `RQ2_review_dynamics.ipynb`
   - `RQ3_acceptance_prediction.ipynb`

# 6. Troubleshooting

Common issues and solutions:

- **HuggingFace parquet loading error**  
  Ensure the following packages are updated: `huggingface_hub`, `fsspec`.

- **Notebook kernel crash**  
  Increase the Jupyter I/O buffer limit.

- **Out-of-memory error during loading**  
  Load specific columns using the `columns=[...]` argument.

- **Missing result files**  
  Re-run the final cell of each notebook; figures and tables are saved automatically.
